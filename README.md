# Mayor's Office

Mayor's Office is a Python city management simulator centered around decision-making and resource management. Players take on the role of a newly elected mayor and must guide their city through unexpected challenges while balancing the needs of citizens, the city budget, and infrastructure.

Each year, the mayor responds to breaking news events and makes decisions that affect four key areas of the city: Population, Budget, Approval, and Infrastructure. Every choice has consequences, and the goal is to keep the city thriving for as long as possible.

## Features

- Turn-based city management gameplay
- Randomly generated events with multiple response options
- Four core city statistics:
  - Population
  - Budget
  - Approval
  - Infrastructure
- Annual municipal summaries tracking changes over time
- Multiple ways to succeed or fail based on player decisions
- Dynamic event outcomes that shape the future of the city

## How to Run

1. Make sure you have Python installed.
2. Clone this repository or download the project files.
3. Open a terminal in the project folder.
4. Run:

```bash
python3 main.py
```
